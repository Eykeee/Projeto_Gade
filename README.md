# 👕 GADE Camiseteria

## 📌 Sobre o Projeto
O **GADE Camiseteria** é um sistema web desenvolvido como projeto de conclusão de curso técnico em informática. A aplicação tem como foco a venda de camisetas personalizadas, oferecendo ao cliente diferentes opções de customização de forma simples e acessível.

O projeto foi baseado em uma empresa real, atuante no mercado desde 2015, o que permitiu alinhar o sistema com necessidades reais do negócio.

---

## 🎯 Objetivo

### Objetivo Geral
Desenvolver um sistema web que possibilite a venda de camisetas personalizadas, permitindo ao usuário escolher estilos, designs e personalizações, facilitando o processo de compra.

### Objetivos Específicos
- Auxiliar na divulgação da marca;
- Melhorar a experiência do cliente no processo de compra;
- Gerenciar usuários (cliente e administrador);
- Permitir a venda de produtos personalizados;

---

## 💡 Motivação
Com o crescimento do uso da tecnologia no comércio, surgiu a necessidade de digitalizar o processo de vendas da empresa GADE. O sistema foi desenvolvido para expandir o alcance do negócio, melhorar o marketing e facilitar a interação com os clientes.

Além disso, o projeto teve como base uma empresa já existente, o que trouxe mais realismo e aplicabilidade prática ao desenvolvimento.

---

## ⚙️ Como Funciona
O sistema permite que usuários:

- Naveguem pelos produtos disponíveis;
- Personalizem camisetas de acordo com suas preferências;
- Realizem pedidos de forma prática;

Além disso, há um sistema de administração que permite:

- Gerenciar produtos;
- Controlar usuários;
- Acompanhar vendas;

---

## 🧠 Metodologia
O desenvolvimento do projeto foi baseado em:

- **Pesquisa qualitativa**
- **Pesquisa de campo**, utilizando dados reais da empresa para entender necessidades e melhorar a solução proposta
